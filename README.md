# EcoBlissBath

- This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.3.0.
- Use __Cypress__ for testing interface and API.
- Use __Symfony__ for API and __MySql__ for DataBase.

>## Docker Backend
### Start
```
docker-compose up -d
```
### Stop :
```
docker-compose down

```

>## FrontEnd
> ### First Action :
```
npm install
```
### Angular start :
```
npm start
```
