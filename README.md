# EcoBlissBath

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.3.0.

## FrontEnd:

### Angular start :
---
Run `npm start`
---
### Cypress Start :
---
Run `npx cypress run`
---


## Docker Backend :
### Start
---
Run `docker-compose up -d`
---
### Stop :
---
Run `docker-compose down`
---
